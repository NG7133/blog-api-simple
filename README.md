#API Blog Simple
## Description
Cette application est une API backend permettant de gerer des articles de blog.

## Technologies
- Node.js
- Express
- HTML / CSS /JavaScript

## Installation
1. Installer les dependances :
   npm install
2. Lancer le serveur :
   node server.js
## Endpoints
-GET/api/articles -> recuperer les articles
-POST /api/articles -> ajouter un article

## Utilisation
ouvrir le navigateur :
http://localhost:3000
